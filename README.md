# mikutter-for-iOS
mikutterの公式アプリです。  
mikutterの最新情報を得ることができます。

## 機能
* FAQ  
    mikutter公式サイトのFAQを確認する機能です。

* お知らせ  
    mikutterのリリース・イベント・グッズ情報などを受け取ることができます。

## 必要要件
|name       |version        |
|:----------|:--------------|
|iOS        |9.3 or higher  |
|XCode      |8.3.2          |
